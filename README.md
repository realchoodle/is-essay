# Readme File

This repository houses my IS paper. The most important files are:

- `is-essay.pdf`, which is the rendered PDF file.
- `is-essay.tex`, which is the raw LaTeX code.
- `bib.bib`, which is my BibLaTeX bibliography.

Clicking on commit messages (for example, "update readme" or "essay progress")
lets you see the specific changes made in that commit. These don't work for PDF
files because they are not a plain text format. However, they do work nicely
for `.tex` files.
